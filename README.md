
## Installation

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/orabitbol/OX.git
   cd OX
   ```

2. **Install Dependencies Backend**:

   ```cd server
   npm install
   ```
   npm start
   
   

3. **Start the Application**:

   ```cd client
   npm run dev
   ```

4. **Open the Application**:
   Navigate to `http://localhost:5173/` in your browser.
